# 42 São Paulo - Piscine

This repository contains exercises developed during the **42 São Paulo Piscine**, an intensive 26-day programming bootcamp focused on C language and low-level computing fundamentals.

---

## About the Piscine

The Piscine is a highly immersive program where students learn programming through peer-to-peer collaboration, problem-solving, and hands-on projects — without traditional lectures.

---

## Repository Structure

### Shell

* **shell00** – Introduction to terminal and basic commands
* **shell01** – File manipulation, permissions, and basic scripting

### C Language

* **C01** – Variables, loops, and basic logic
* **C02** – Strings and character manipulation
* **C03** – Functions and code modularization
* **C04** – Arguments, arrays, and more advanced concepts
* **C05** – Recursion and problem-solving techniques

---

## Objective

The goal of this repository is to demonstrate:

* Programming logic and algorithmic thinking
* Strong understanding of C fundamentals
* Ability to solve problems under constraints
* Autonomy in learning and debugging

---

## 42 Rules

All exercises follow the constraints imposed by 42:

* ❌ No use of unauthorized functions
* ❌ No external libraries
* ✅ Use of low-level functions like `write`
* ✅ Focus on manual implementation and understanding

---

## Key Learnings

* Memory manipulation
* Pointer usage
* Control structures (loops and conditionals)
* Code organization and readability
* Problem-solving without relying on high-level abstractions
